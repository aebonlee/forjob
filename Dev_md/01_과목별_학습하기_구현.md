# ForJob - 과목별 학습하기 구현 문서

**작성일:** 2026-04-05
**프로젝트:** ForJob (직업상담사 2급 시험 준비 플랫폼)
**배포 URL:** https://forjob.dreamitbiz.com

---

## 개요

ForJob 사이트에 **과목별 학습 콘텐츠** 페이지를 추가했습니다.
기존에는 CBT 문제풀이(필기)와 서술형 연습(실기)만 있었으나, 과목별로 정리된 이론/개념 학습 페이지를 신규 구현했습니다.

## 라우팅

```
/learn                    LearnHome (6개 과목 카드 허브)
/learn/:subjectCode       SubjectStudy (과목별 챕터/토픽 아코디언)
```

### subjectCode 값
- `counseling` - 직업상담학
- `psychology` - 직업심리학
- `jobinfo` - 직업정보론
- `labor_market` - 노동시장론
- `labor_law` - 노동관계법규
- `silgi` - 실기 (직업상담 실무)

## 데이터 구조

`src/data/curriculum.js`의 구조:

```js
CURRICULUM = {
  [subjectCode]: {
    name: string,       // 과목명
    code: string,       // 과목코드
    color: string,      // 테마 색상 (#hex)
    icon: string,       // FontAwesome 아이콘 클래스
    description: string,// 과목 설명
    chapters: [
      {
        title: string,  // 챕터 제목
        topics: [
          {
            title: string,   // 토픽 제목
            content: string, // HTML 형식 본문
          }
        ]
      }
    ]
  }
}
```

| 과목 | 챕터 수 | 토픽 수 |

| 직업심리학 | 4 | 7 |
| 직업정보론 | 4 | 5 |
| 노동시장론 | 4 | 4 |
| 노동관계법규 | 5 | 5 |

## UI/UX 설계

### LearnHome (`/learn`)
- 6개 과목 카드 그리드 (2열 → 1열 반응형)
- 각 카드: 아이콘 + 과목명 + 설명 + 챕터/토픽 수 배지
- 클릭 시 해당 과목 학습 페이지로 이동

### SubjectStudy (`/learn/:subjectCode`)
- 2열 레이아웃: 사이드바(목차) + 메인(콘텐츠)
- 사이드바: sticky 챕터 네비게이션 (데스크탑만)
- 메인: 챕터 아코디언 → 토픽 아코디언 (2단계)
- 토픽 본문: HTML 렌더링 (테이블, 리스트, 공식박스 등)
- 하단: 이전/다음 과목 네비게이션

### 네비게이션
- **데스크탑:** "학습하기" hover 시 6개 과목 드롭다운
- **모바일:** "학습하기" 클릭 시 서브내비 아코디언 펼침
