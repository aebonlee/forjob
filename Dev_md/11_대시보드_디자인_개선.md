**작성일:** 2026-04-05
**프로젝트:** ForJob (직업상담사 2급 시험 준비 플랫폼)

## 개요

학습 대시보드의 마진, 패딩, 누락 스타일을 정리하여 전체적으로 보기 좋게 개선했습니다.

---

## 변경 사항

### 1. 누락 CSS 클래스 추가 (7개)

컴포넌트에서 사용하지만 CSS에 정의되지 않았던 클래스들:

| 클래스 | 용도 |
|--------|------|
| `.dashboard-stats-grid` | 통계 카드 4열 그리드 (기존 `.stat-cards`와 분리) |
| `.stat-icon` | 통계 카드 아이콘 (44px, 라운드, 파란 배경) |
| `.dashboard-empty` | "시험 기록이 없습니다" 빈 상태 표시 |
| `.dashboard-good` | "모든 과목이 60점 이상" 성공 상태 표시 |
| `.history-table-wrapper` | 테이블 overflow-x 래퍼 + 라운드 코너 |
| `.table-badge` | 합격/불합격 테이블 내 작은 뱃지 (.pass/.fail) |
| `.prediction-empty` | 합격 예측 빈 상태 표시 |

### 2. 합격 예측 카드 스타일 (5개 클래스)

PassPrediction.tsx에서 사용하는 스타일:

| 클래스 | 설명 |
|--------|------|
| `.prediction-circle` | 128px 원형 확률 표시 (high/medium/low 컬러) |
| `.prediction-percent` | 36px 굵은 퍼센트 텍스트 |
| `.prediction-stats` | 3열 통계 그리드 (최근 평균, 합격률, 응시 횟수) |
| `.prediction-stat-label` | 12px 라벨 |
| `.prediction-stat-value` | 16px 볼드 값 |

### 3. 패딩/마진 개선

| 요소 | 변경 전 | 변경 후 |
|------|---------|---------|
| `.stat-card` padding | 24px | 28px |
| `.dashboard-card` padding | 28px | 28px |
| `.dashboard-grid` gap | 24px | 28px |
| `.dashboard-grid` margin-top | 없음 | 28px |
| `.weakness-item` padding | 12px 16px | 14px 18px |

### 4. 카드 헤더 디자인

`.dashboard-card h3`에 하단 구분선 추가:
- `padding-bottom: 16px`
- `border-bottom: 1px solid var(--border-light)`
- 아이콘 색상: `var(--primary-blue)`

### 5. 최근 시험 헤더

인라인 스타일 → `.dashboard-recent-header` CSS 클래스로 교체.
h3 중복 border-bottom 제거 처리.

### 6. 약점 과목 리스트

`.weakness-item`에 `justify-content: space-between` 추가 — 과목명과 점수가 양 끝 정렬.

### 7. 반응형 처리

`.dashboard-stats-grid` 반응형 규칙 추가:
- 1023px 이하: 2열
- 767px 이하: 2열
- 639px 이하: 1열
